HTML Tags

<section></section>
This tag is used to group logically related content and create page sections into logical blocks (news block, contact information, etc.)
<nav></nav>
A section of a document that contains navigational links throughout a site.
<blockquote></blockquote>
This tag indicates that the enclosed text is an extended quotation.
<cite></cite>
It is a link to the source of the quote (books, articles, poems, scripts, films, songs, operas, games). It must include the title of the work or URL, which can be abbreviated.
<q></q>
This element is used to highlight quotations in the text. The content of the container is automatically displayed in the browser in quotation marks.
